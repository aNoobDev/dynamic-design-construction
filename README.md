# Dynamic Design & Construction

A fast, responsive static marketing website for a construction and design studio.

## Project structure

- index.html — main page entry point
- src/styles/main.css — site styling
- src/scripts/main.js — lightweight JavaScript initialization
- src/assets/ — future images and media assets

## Maintainability and security notes

- Keep the site simple and static for easier maintenance.
- Store styles in the CSS file and keep selectors organized.
- Keep JavaScript small, focused, and progressively enhanced.
- Avoid inline scripts and inline styles where possible.
- For future growth, consider a static-site generator or a lightweight build step.
